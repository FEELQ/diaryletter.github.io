🔒 DiaryLetter プライバシーポリシー

最終更新日: 2025年8月6日
💌 DiaryLetterは、ユーザーの個人情報を大切にし、個人情報保護法に従って安全に保護しています。

1. 個人情報の処理目的

DiaryLetterは以下の目的のために個人情報を処理します：
  • 会員登録および本人識別・認証
  • 日記作成および保存サービスの提供
  • AI手紙生成サービスの提供
  • 個人化されたユーザー体験の提供
  • サービス改善およびカスタマーサポート
  • 日記作成通知などのカスタマイズサービスの提供

2. 処理する個人情報の項目

DiaryLetterは以下の個人情報項目を処理します：

📧 会員情報

  • メールアドレス（必須）
  • 暗号化されたパスワード（メール登録時）
  • ソーシャルログイン識別子（ソーシャルログイン時）

📝 日記関連情報

  • 日記のタイトルおよび内容
  • 感情状態（嬉しい、悲しい、怒り、不安、ときめきなど）
  • 天気情報（晴れ、曇り、雨、雪）
  • 社会的状況（一人、家族、友達、恋人）
  • 活動タイプ（仕事、余暇、休息、運動、自己啓発、旅行、日常）
  • 日記作成日時

🤖 AIサービス関連

  • AIが生成した手紙内容
  • 手紙生成履歴

📱 アプリ使用情報

  • テーマおよびフォント設定
  • 通知設定
  • アプリ使用統計（個人識別不可能な形式）

3. 個人情報の処理および保有期間

DiaryLetterは以下のように個人情報を処理・保有します：
  • 会員情報： 会員退会まで
  • 日記および手紙内容： ユーザーが直接削除するか会員退会まで
  • アプリ使用設定： 会員退会まで

ただし、以下の場合には該当期間まで保管します：

  • 関連法令違反による捜査・調査などが進行中の場合：該当捜査・調査終了まで

4. 個人情報の第三者提供

DiaryLetterは原則として個人情報を第三者に提供しません。

ただし、以下の場合には例外とします：

  • ユーザーが事前に同意した場合
  • 法令の規定により、または捜査目的で法令に定められた手続きと方法に従って捜査機関の要求がある場合

5. 個人情報処理の委託

DiaryLetterは円滑なサービス提供のため、以下のように個人情報処理業務を委託しています：

☁️ Supabase Inc.

  • 委託業務： データ保存およびユーザー認証
  • 個人情報保有期間： 委託契約終了時または委託目的達成時まで

🤖 Google LLC

  • 委託業務： AI手紙生成サービス
  • 個人情報保有期間： 手紙生成後即座に削除（保存しません）

委託業者は個人情報保護関連法令を遵守し、委託目的以外の個人情報処理を禁止しています。

6. 情報主体の権利・義務およびその行使方法

ユーザーはいつでも以下のような個人情報保護関連権利を行使できます：
  • 個人情報閲覧要求
  • 個人情報訂正・削除要求
  • 個人情報処理停止要求

権利行使は以下の方法で可能です：

  • アプリ内設定メニューを通じた直接修正/削除
  • メール（hpil331@gmail.com）による要求

要求時は本人確認手続きを経て、法令に定められた期間内に処理いたします。

DiaryLetterは個人情報保護のため以下のような措置を講じています：

🔧 技術的措置

  • 個人情報の暗号化保存
  • 安全なデータ転送のためのSSL/TLS適用
  • ハッキング防止のためのセキュリティプログラム設置および点検

📋 管理的措置

  • 個人情報アクセス権限の最小化
  • 定期的なセキュリティ点検およびアップデート

  • データセンターの物理的セキュリティ施設
